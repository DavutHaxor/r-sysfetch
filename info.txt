CPU Info:

· /proc/stat - CPU usage percentages
· /proc/cpuinfo - CPU model, cores
· /proc/loadavg - system load

GPU Info:

. /sys/class/drm/card1/ 
. /sys/class/drm/card2/ 

Memory Info:

· /proc/meminfo - total, free, cached memory
· /proc/vmstat - virtual memory statistics

Process Info:

· /proc/pid/stat - individual process stats
· /proc/pid/status - process memory usage
· /proc/pid/io - process I/O statistics
. 

cpu template: r-sysfetch cpu {flags}
flags:
/proc/cpuinfo model name = m 
/proc/cpuinfo cores = c 
/proc/cpuinfo siblings (threads) = t
/sys/class/hwmon/hwmonX/temp1_input = h
/proc/stat/ usage = u
how to calculate usage: read /proc/stat in 2 different times, 4th number is idle
usage% = [(total_diff) - (idle_diff)] * 100 / total_diff 
/sys/devices/system/cpu/cpu*/cpufreq/scaling_cur_freq speed = s
